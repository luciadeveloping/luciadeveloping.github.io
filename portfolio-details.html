Bouba District

Project information

  • Project date: September-February, 2024
  • Languajes & Technologies: Unity, WebGL, C#
  • Project URL: www.example.com

Description

A satisfying rhythm game where you face cute slimy creatures. The game can be played on the web from any device. There’s a global scoreboard too! It’s available on Itchio: https://syncstudios-games.itch.io/bouba-district

Comments

This game was developed through the course of three months. From the start my team and I tried to be realistic with our expectations and I think we achieved good estimations for the game’s requirements. The goal was less complexity but responsive and satisfying gameplay. I worked on the core gameplay features as well as the user experience. I would’ve liked to spend more time on the UX aspect but considering our time limitations I think it turned out really well.

Project name

Project information

  • Project date: September-February, 2023
  • Languajes & Technologies: JavaScript, Phaser, HTML, CSS, Websockets
  • Project URL: www.example.com

Description

A strategic 1v1 game played on the same keyboard. Conquer other cities which generate soldiers which allow you to... conquer more cities!

Comments

I’m quite proud of this one. My team and I made it using the phaser framework, it was challenging but there were great ideas and effort put into this game. Also a big shout out to my teammates for the game’s visuals, while simple I think they look great and pro. Back to the game I really enjoyed working on this and in the end I think my perspective on javascript as a language changed dramatically. I followed up on what I learnt in my previous project Space Invaders since it was basically the same thing: a game from scratch (only now I had phaser on my side). Fast forward into the future we upgraded the game and added local multiplayer (no more awkward controllers for player 2) using websockets. It was my first contact with multiplayer and I enjoyed implementing it. The final version was never published but it’s accessible via the github link.

Steam War (Guerra de vapor)

Project information

  • Project date: September-February, 2022
  • Languajes & Technologies: Unity, C#, Xbox Adaptive Controller
  • Project URL: www.example.com

Description

A strategic card game where you play as a machine that uses its health as a resource to fight scary enemies. Play with only two buttons!

Comments

This game was made quite early in my college life. I didn’t have much experience back then but this was one of my first “complex” games I made on the Unity editor. The gameplay sure feels confusing and not very responsive but what I think stands out about this project was the design choices. The game was made for a class about accessibility and it was meant to be played on the Xbox Accessible Controller. We were limited to two buttons only for the gameplay and that turned out to be a fun challenge. I took the role of game designer and implemented most of the game features.
